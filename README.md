# File Storage Test

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)
[![Storage](https://img.shields.io/badge/Feature-File_Storage-blue.svg)](https://developer.android.com/training/data-storage)

Тестовое приложение для изучения работы с файловым хранилищем в Android. Демонстрирует чтение и запись файлов во внешнее хранилище.

## Функциональность

- Сохранение текста в файл на внешнем хранилище
- Чтение данных из файла
- Проверка доступности внешнего хранилища
- Обработка ошибок ввода-вывода

## Технологии

- Kotlin
- File I/O operations
- External storage
- FileInputStream/FileOutputStream
- Environment checks
